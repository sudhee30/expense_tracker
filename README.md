# Expense Tracker

## Overview

Expense Tracker is a web application designed to manage and track personal finances. It allows users to input and categorize income and expenses, view a summary of their financial status, and visualize their data with a pie chart. The application also includes functionality to save and load data for different months and features a responsive design with a gradient background animation.

## Features

- **Add New Entry:** Allows users to input details for income or expenses.
- **Financial Summary:** Displays total income, total expenses, and current balance.
- **Pie Chart Visualization:** Uses Chart.js to represent income and expenses dynamically.
- **Save and Load Data:** Save your financial data for a selected month and load it later.
- **Dynamic Background:** Animated gradient background that responds to mouse movement.

## Technologies Used

- **HTML/CSS:** Structure and styling of the web application.
- **JavaScript:** Dynamic functionality, including data management and chart updates.
- **Chart.js:** Library for creating interactive pie charts.

## Installation

To get started with Expense Tracker locally:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/sudhee30/expense_tracker.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd expense_tracker/exp%202
    ```

3. **Open `index.html` in your web browser:**

    You can open the file directly or use a local server to view it.

## Usage

1. **Add New Entry:**
   - Fill in the entry details (Name, Amount, Type) and click "Add Entry."
   
2. **View Summary:**
   - The summary section will update to show total income, total expenses, and balance.
   - The pie chart will reflect the current financial data.

3. **Save and Load Data:**
   - Select a month from the dropdown menu and use the "Save Data" or "Load Data" buttons to manage your data for that month.
